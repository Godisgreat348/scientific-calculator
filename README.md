Daniel Scientific Calculator
A full-featured calculator app built as a single self-contained web app (HTML/CSS/JavaScript). Runs in any browser, desktop or mobile with no installation or build step required.
Live Demo
Open scientific-calculator.html directly in a browser, or enable GitHub Pages for this repo to get a hosted link.
Features
Core (Basic Mode)
Addition, subtraction, multiplication, division
Percentage, sign toggle, decimal support, backspace, clear
Bonus Features
Scientific Mode
Trigonometric functions: sin, cos, tan, asin, acos, atan
Hyperbolic functions: sinh, cosh, tanh
Logarithms: natural log (ln), base-10 log (log)
Powers and roots: square, cube, square root, cube root, general exponent (^)
Factorial, reciprocal (1/x), constants π and e
Degree / Radian toggle for angle-based functions
Matrix Mode
Supports 2×2, 3×3, and 4×4 matrices
Operations: addition, subtraction, multiplication, determinant, inverse, transpose
Permutations & Combinations Mode
nPr (permutations), nCr (combinations), n! (factorial)
Statistics Mode
Accepts a comma/space-separated data set
Computes: count, sum, mean, median, mode, range, min, max
Variance and standard deviation (both population and sample)
Tech Stack
Vanilla HTML, CSS, and JavaScript, no frameworks or external dependencies (aside from Google Fonts)
Single file (scientific-calculator.html), easy to open or deploy anywhere
How to Run
Download scientific-calculator.html
Open it in any modern browser (Chrome, Firefox, Safari, Edge)
Use the top navigation tabs to switch between Basic, Scientific, Matrix, nPr/nCr, and Statistics modes
Notes
Built to satisfy the assignment's core requirement (basic arithmetic operations) plus all listed bonus categories: trigonometric functions, hyperbolic functions, matrix operations, permutations/combinations, and statistical computations.
